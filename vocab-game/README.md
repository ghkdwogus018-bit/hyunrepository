# 단어팡 · 심슨보카 하루 복습 🐣

하루치 영어 단어를 게임하듯 가볍게 복습하는 웹 앱입니다.
영어 단어가 하나씩 나오면 아래 **1~4번 보기** 중 뜻을 골라 정답을 맞히고,
맞히면 귀여운 이펙트(콘페티·콤보·XP)가 터지고, 틀린 단어는 자동으로 **복습 목록**에 담깁니다.

교재: **심슨보카 2027 · DAY별 주관식 테스트** (총 48일차 / 2,400단어, 첨부 PDF에서 추출)

## 주요 기능

- **Day 선택** — 48일치 중 원하는 날을 골라 학습 (문제 수 10 / 25 / 전체 선택)
- **4지선다 터치 퀴즈** — 같은 품사끼리 그럴듯한 오답 보기를 자동 구성
- **정답 이펙트** — 콘페티, 콤보 연출, 마스코트 반응, 부드러운 효과음, XP 획득
- **오답 복습** — 틀린 단어는 정답을 바로 보여주고 복습 목록에 저장, 나중에 다시 풀기
- **성장 요소** — 레벨/XP, 연속 학습(스트릭 🔥), Day별 완료·별점 기록
- **키보드 지원** — PC에서는 숫자키 1~4로도 정답 선택
- **오프라인 저장** — 모든 기록은 브라우저 `localStorage`에만 저장 (서버 불필요)

## 실행 방법

정적 파일이라 서버 없이도 열립니다.

```bash
# 방법 1: 파일을 직접 열기
#   vocab-game/index.html 을 브라우저로 열기

# 방법 2: 간단한 로컬 서버 (권장)
cd vocab-game
python3 -m http.server 8000
#   → http://localhost:8000 접속
```

모바일에서 브라우저 메뉴의 "홈 화면에 추가"를 하면 앱처럼 전체화면으로 쓸 수 있어요.

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 + 스타일 (자체 포함) |
| `app.js` | 게임 로직 (퀴즈·이펙트·기록, 바닐라 JS, 외부 의존성 없음) |
| `data.js` | 단어 데이터 (`window.VOCAB`, PDF에서 추출한 48일 × 50단어) |

## 데이터 형식

```js
window.VOCAB = [
  { day: 1, words: [
    { no: 1, word: "susceptible", meaning: "a. (병에) 걸리기 쉬운, 취약한 …", pos: "a" },
    …
  ]},
  …
];
```
